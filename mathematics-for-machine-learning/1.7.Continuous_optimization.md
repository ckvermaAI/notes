## 7.1 Gradient Descent

For $\min_{\boldsymbol{x}} f(\boldsymbol{x})$, $f: \mathbb{R}^d \to \mathbb{R}$ differentiable:
$$\boldsymbol{x}_{t+1} = \boldsymbol{x}_t - \gamma_t (\nabla f(\boldsymbol{x}_t))^T$$

Gradients are row vectors by convention; the update uses the transpose. For small enough $\gamma_t$, $f(\boldsymbol{x}_{t+1}) \leq f(\boldsymbol{x}_t)$.

**Step size**: too small → slow; too large → divergence/oscillation. Adaptive strategies: reduce $\gamma$ on function increase, increase on decrease. In practice: line search, Armijo condition, or schedule-based (e.g., cosine annealing).

### Momentum

Adds memory of past updates, smoothing oscillations in ill-conditioned problems (elongated loss surfaces):
$$\boldsymbol{x}_{t+1} = \boldsymbol{x}_t - \gamma_t (\nabla f(\boldsymbol{x}_t))^T + \alpha \Delta\boldsymbol{x}_t$$

where $\alpha \in [0,1]$ and $\Delta\boldsymbol{x}_t = \boldsymbol{x}_t - \boldsymbol{x}_{t-1}$. Equivalent to a heavy-ball method: the exponential moving average of past gradients. Nesterov momentum adds a lookahead correction for better convergence guarantees.

### Stochastic Gradient Descent (SGD)

Full batch $\nabla L(\boldsymbol{\theta}) = \sum_n \nabla L_n(\boldsymbol{\theta})$ is $O(N)$ per step. SGD uses a mini-batch subset:
$$\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t - \gamma_t \sum_{n \in \mathcal{B}} \nabla L_n(\boldsymbol{\theta}_t)$$

The mini-batch gradient is an **unbiased estimator** of the full gradient. Converges under diminishing step-size schedule ($\sum_t \gamma_t = \infty$, $\sum_t \gamma_t^2 < \infty$). Noise in SGD can help escape poor local minima and saddle points; small batches are noisier, large batches approximate full-batch GD.

## 7.2 Constrained Optimization: Lagrange Multipliers

For $\min_{\boldsymbol{x}} f(\boldsymbol{x})$ s.t. $g_i(\boldsymbol{x}) \leq 0$, $h_j(\boldsymbol{x}) = 0$:

**Lagrangian**:
$$\mathfrak{L}(\boldsymbol{x}, \boldsymbol{\lambda}, \boldsymbol{\mu}) = f(\boldsymbol{x}) + \sum_i \lambda_i g_i(\boldsymbol{x}) + \sum_j \mu_j h_j(\boldsymbol{x})$$

with $\lambda_i \geq 0$ (inequality) and $\mu_j \in \mathbb{R}$ (equality).

**Dual function**: $\mathfrak{D}(\boldsymbol{\lambda}, \boldsymbol{\mu}) = \min_{\boldsymbol{x}} \mathfrak{L}(\boldsymbol{x}, \boldsymbol{\lambda}, \boldsymbol{\mu})$. The dual problem $\max_{\boldsymbol{\lambda} \geq 0, \boldsymbol{\mu}} \mathfrak{D}(\boldsymbol{\lambda}, \boldsymbol{\mu})$ is always convex (even if the primal is not).

**Weak duality**: $\mathfrak{D}^* \leq f^*$ (dual optimal $\leq$ primal optimal). **Strong duality** (primal = dual): holds under Slater's condition for convex problems (see §7.3).

**KKT conditions** (necessary at optimality for smooth problems, sufficient for convex):
1. Stationarity: $\nabla_{\boldsymbol{x}} \mathfrak{L} = \mathbf{0}$
2. Primal feasibility: $g_i(\boldsymbol{x}^*) \leq 0$, $h_j(\boldsymbol{x}^*) = 0$
3. Dual feasibility: $\lambda_i^* \geq 0$
4. Complementary slackness: $\lambda_i^* g_i(\boldsymbol{x}^*) = 0$ — either $\lambda_i = 0$ or the constraint is active.

## 7.3 Convex Optimization

**Convex set**: $\theta\boldsymbol{x} + (1-\theta)\boldsymbol{y} \in \mathcal{C}$ for all $\boldsymbol{x}, \boldsymbol{y} \in \mathcal{C}$, $\theta \in [0,1]$.

**Convex function**: $f(\theta\boldsymbol{x} + (1-\theta)\boldsymbol{y}) \leq \theta f(\boldsymbol{x}) + (1-\theta)f(\boldsymbol{y})$. Equivalently (for differentiable $f$): $f(\boldsymbol{y}) \geq f(\boldsymbol{x}) + \nabla f(\boldsymbol{x})^T(\boldsymbol{y}-\boldsymbol{x})$ (first-order characterization); for twice-differentiable: $\nabla^2 f(\boldsymbol{x}) \succeq 0$ (PSD Hessian).

For convex $f$ and constraints, every local minimum is a global minimum. Non-negative weighted sums of convex functions are convex. Composition rules: $g(f(\boldsymbol{x}))$ is convex if $g$ is convex nondecreasing and $f$ is convex.

### Linear Programming

$\min_{\boldsymbol{x}} \boldsymbol{c}^T\boldsymbol{x}$ s.t. $\boldsymbol{A}\boldsymbol{x} \leq \boldsymbol{b}$. Dual: $\max_{\boldsymbol{\lambda}} -\boldsymbol{b}^T\boldsymbol{\lambda}$ s.t. $\boldsymbol{c} + \boldsymbol{A}^T\boldsymbol{\lambda} = \mathbf{0}$, $\boldsymbol{\lambda} \geq \mathbf{0}$. Strong duality holds (LP duality).

### Quadratic Programming

$\min_{\boldsymbol{x}} \frac{1}{2}\boldsymbol{x}^T\boldsymbol{Q}\boldsymbol{x} + \boldsymbol{c}^T\boldsymbol{x}$ s.t. $\boldsymbol{A}\boldsymbol{x} \leq \boldsymbol{b}$, $\boldsymbol{Q} \succ 0$. The SVM primal/dual formulation is a QP (see Chapter 12). Solution via $\boldsymbol{x}^* = -\boldsymbol{Q}^{-1}(\boldsymbol{c} + \boldsymbol{A}^T\boldsymbol{\lambda}^*)$.

### Legendre-Fenchel (Convex Conjugate)

The convex conjugate of $f$:
$$f^*(\boldsymbol{s}) = \sup_{\boldsymbol{x}} \big(\langle \boldsymbol{s}, \boldsymbol{x} \rangle - f(\boldsymbol{x})\big)$$

Properties: $f^*$ is always convex (even if $f$ is not); $(f^*)^* = f$ for convex $f$ (Fenchel duality). Key use: converting primal problems to dual forms (e.g., SVM, kernel ridge regression). For $f(\boldsymbol{y}) = \frac{\lambda}{2}\boldsymbol{y}^T\boldsymbol{K}^{-1}\boldsymbol{y}$: $f^*(\boldsymbol{\alpha}) = \frac{1}{2\lambda}\boldsymbol{\alpha}^T\boldsymbol{K}\boldsymbol{\alpha}$ — this is how kernelized ridge regression dual arises.
