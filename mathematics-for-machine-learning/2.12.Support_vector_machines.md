# Support Vector Machines

*Source: Mathematics for Machine Learning, Deisenroth, Faisal, Ong (2020), Chapter 12*

---

## 12.1 Separating Hyperplanes

Binary classification: predict $y \in \{+1, -1\}$ from $\boldsymbol{x} \in \mathbb{R}^D$.

A hyperplane is $\langle \boldsymbol{w}, \boldsymbol{x} \rangle + b = 0$. The decision function:
$$f(\boldsymbol{x}) = \langle \boldsymbol{w}, \boldsymbol{x} \rangle + b$$

classifies as $+1$ if $f(\boldsymbol{x}) \geq 0$ and $-1$ otherwise. The normal vector $\boldsymbol{w}$ is orthogonal to the hyperplane: for any $\boldsymbol{x}_a, \boldsymbol{x}_b$ on the plane, $\langle \boldsymbol{w}, \boldsymbol{x}_a - \boldsymbol{x}_b \rangle = 0$.

Training constraint: $y_n(\langle \boldsymbol{w}, \boldsymbol{x}_n \rangle + b) \geq 0$ for all $n$.

---

## 12.2 Primal SVM

### Margin

Among all separating hyperplanes, SVMs select the one maximizing the **margin** $= 2/\|\boldsymbol{w}\|$, which is the distance between the support hyperplanes $\langle \boldsymbol{w}, \boldsymbol{x} \rangle + b = \pm 1$.

By the rescaling argument (Theorem 12.1), maximizing margin is equivalent to the **hard margin SVM**:
$$\min_{\boldsymbol{w}, b} \frac{1}{2}\|\boldsymbol{w}\|^2 \quad \text{s.t.} \quad y_n(\langle \boldsymbol{w}, \boldsymbol{x}_n \rangle + b) \geq 1 \;\forall n$$

Maximizing margin improves generalization: larger margin $\Leftrightarrow$ lower Vapnik-Chervonenkis dimension $\Leftrightarrow$ tighter generalization bound.

### Soft Margin SVM

For non-linearly separable data, introduce slack variables $\xi_n \geq 0$:
$$\min_{\boldsymbol{w}, b, \boldsymbol{\xi}} \frac{1}{2}\|\boldsymbol{w}\|^2 + C\sum_n \xi_n \quad \text{s.t.} \quad y_n(\langle \boldsymbol{w}, \boldsymbol{x}_n \rangle + b) \geq 1 - \xi_n, \; \xi_n \geq 0$$

$C > 0$ trades off margin size vs. training error. Large $C$ → hard margin (low bias, high variance); small $C$ → large margin (high bias, low variance). Note $\boldsymbol{w}$ is regularized; $b$ is not.

### Hinge Loss View

The soft margin SVM is equivalent to ERM with the **hinge loss** $\ell(t) = \max(0, 1-t)$:
$$\min_{\boldsymbol{w}, b} \frac{1}{2}\|\boldsymbol{w}\|^2 + C\sum_n \max(0, 1 - y_n(\langle \boldsymbol{w}, \boldsymbol{x}_n \rangle + b))$$

The $\frac{1}{2}\|\boldsymbol{w}\|^2$ term is the $L_2$ regularizer (equivalent to Gaussian prior from the MAP perspective).

## 12.3 Dual SVM

The Lagrangian of the primal soft margin SVM:
$$\mathfrak{L}(\boldsymbol{w}, b, \boldsymbol{\xi}, \boldsymbol{\alpha}, \boldsymbol{\gamma}) = \frac{1}{2}\|\boldsymbol{w}\|^2 + C\sum_n\xi_n - \sum_n\alpha_n(y_n(\langle\boldsymbol{w},\boldsymbol{x}_n\rangle + b) - 1 + \xi_n) - \sum_n\gamma_n\xi_n$$

KKT stationarity conditions:
- $\nabla_{\boldsymbol{w}}\mathfrak{L} = 0 \Rightarrow \boldsymbol{w} = \sum_n \alpha_n y_n \boldsymbol{x}_n$ (**representer theorem**)
- $\nabla_b\mathfrak{L} = 0 \Rightarrow \sum_n \alpha_n y_n = 0$
- $\nabla_{\xi_n}\mathfrak{L} = 0 \Rightarrow \alpha_n = C - \gamma_n \leq C$ (since $\gamma_n \geq 0$)

Substituting back gives the **dual QP**:
$$\min_{\boldsymbol{\alpha}} \frac{1}{2}\sum_{i,j} y_i y_j \alpha_i \alpha_j \langle \boldsymbol{x}_i, \boldsymbol{x}_j \rangle - \sum_i \alpha_i \quad \text{s.t.} \quad \sum_n y_n \alpha_n = 0,\; 0 \leq \alpha_n \leq C$$

**Support vectors**: examples with $\alpha_n > 0$. From complementary slackness:
- $0 < \alpha_n < C$: $\xi_n = 0$, $\boldsymbol{x}_n$ on the margin boundary
- $\alpha_n = C$: $\xi_n > 0$, $\boldsymbol{x}_n$ inside or beyond the margin

Bias $b^*$ recovered from any support vector on the margin: $b^* = y_n - \langle \boldsymbol{w}^*, \boldsymbol{x}_n \rangle$.

**Why dual?** The dual scales with $N$ (not $D$), advantageous when $D \gg N$. More importantly, it exposes the inner product structure enabling kernelization.

### Geometric (Convex Hull) View

The hard margin SVM finds the minimum-norm vector $\boldsymbol{w} = \boldsymbol{c} - \boldsymbol{d}$ where $\boldsymbol{c} \in \text{conv}(\{\boldsymbol{x}_n : y_n = +1\})$ and $\boldsymbol{d} \in \text{conv}(\{\boldsymbol{x}_n : y_n = -1\})$ are the closest points of the respective convex hulls.

## 12.4 Kernels

The dual depends on data only through inner products $\langle \boldsymbol{x}_i, \boldsymbol{x}_j \rangle$. The **kernel trick** replaces these with $k(\boldsymbol{x}_i, \boldsymbol{x}_j) = \langle \phi(\boldsymbol{x}_i), \phi(\boldsymbol{x}_j) \rangle_\mathcal{H}$ for some (possibly infinite-dimensional) feature map $\phi$.

**Valid kernel** $\Leftrightarrow$ the $N \times N$ Gram matrix $\boldsymbol{K}_{ij} = k(\boldsymbol{x}_i, \boldsymbol{x}_j)$ is symmetric positive semidefinite for any input set.

Common kernels:
- **Linear**: $k(\boldsymbol{x}, \boldsymbol{x}') = \langle \boldsymbol{x}, \boldsymbol{x}' \rangle$
- **Polynomial**: $k(\boldsymbol{x}, \boldsymbol{x}') = (\langle \boldsymbol{x}, \boldsymbol{x}' \rangle + c)^d$ — implicitly maps to all monomials up to degree $d$
- **RBF (Gaussian)**: $k(\boldsymbol{x}, \boldsymbol{x}') = \exp(-\|\boldsymbol{x}-\boldsymbol{x}'\|^2 / 2\ell^2)$ — corresponds to infinite-dimensional RKHS

The prediction for a new point uses $f(\boldsymbol{x}_*) = \sum_n \alpha_n y_n k(\boldsymbol{x}_n, \boldsymbol{x}_*) + b$, evaluating the kernel only at support vectors.

## 12.5 Numerical Solution

**Unconstrained** (hinge loss + gradient descent): subgradient $\partial\ell(t) = -1$ if $t < 1$, $0$ if $t > 1$, any value in $[-1, 0]$ at $t = 1$.

**QP solvers**: primal (direct, $O(N^3)$ in general) or dual (coordinate descent on $\alpha$, e.g., SMO algorithm which decomposes into 2-variable subproblems). LIBSVM implements SMO with caching; scales to millions of examples.

## 12.6 Further Reading

- **Theoretical guarantees**: margin-based generalization bounds (Vapnik-Chervonenkis theory); PAC learning. SVMs have one of the tightest classical generalization bounds.
- **Probabilistic outputs**: Platt scaling calibrates SVM scores to probabilities via sigmoid fit on held-out data.
- **Multi-class**: one-vs-one or one-vs-rest decomposition; Crammer-Singer formulation gives a joint multi-class objective.
- **Gaussian process classification**: Bayesian analogue using a GP prior on $f$; posterior is non-Gaussian (requires Laplace approximation or EP).
- **Modern relevance**: SVMs are largely superseded by deep networks for perception tasks but remain competitive on tabular/structured data and in low-data regimes. The kernel perspective is foundational for understanding attention (dot-product attention ≈ kernel regression) and neural tangent kernels.
